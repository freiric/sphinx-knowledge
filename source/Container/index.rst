Vagrant
**********

wrapper around virtual box (or VMware).
install from download page.
`catalogue <https://vagrantcloud.com/boxes/search>`_

.. code-block:: sh

   vagrant init boxname # create a Vagrantfile in current directory
   vagrant up

   vagrant add boxname
